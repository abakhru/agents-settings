# Feed-Analyzer — Social Feed

**check_types**: [social-feed]

## Analysis Criteria

Analyse the screenshot and accessibility tree for:

**Social Feed Issues:**
- Posts not loading in feed
- Infinite scroll not working
- Like/reaction buttons not working
- Comment button broken
- Share button not working
- Post images not loading
- Post timestamps missing or wrong
- Feed filtering not working
- "Load more" broken
- New post indicator not updating
- Feed order incorrect
- Mobile feed display issues
