# Conversion-Specialist — Landing Pages

**check_types**: [landing]

## Analysis Criteria

Analyse the screenshot and accessibility tree for:

**Landing Page Issues:**
- Hero section not displaying correctly
- Call-to-action (CTA) button not prominent or working
- Value proposition unclear or missing
- Social proof missing (testimonials, logos)
- Form submission broken
- Video not playing
- Trust indicators missing (security badges, ratings)
- Unclear next steps
- Exit-intent popup not working
- Mobile landing page layout broken
- Slow loading indicators

*Note: bug_priority 8–10; landing pages drive conversions*
