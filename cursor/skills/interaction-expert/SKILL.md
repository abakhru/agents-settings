# Interaction-Expert — JavaScript & Booking Flows

**check_types**: [javascript, booking]

## Analysis Criteria

Analyse the screenshot, console messages, and accessibility tree for:

**JavaScript Issues:**
- Console errors and warnings
- Uncaught exceptions or promise rejections
- JavaScript runtime errors
- Broken interactive elements due to JS failures
- Event handler issues (clicks not working)
- State management problems

**Booking Flow Issues (if applicable):**
- Date picker problems
- Calendar selection issues
- Booking confirmation errors
- Time slot selection problems
- Reservation form validation issues
- Checkout process problems
