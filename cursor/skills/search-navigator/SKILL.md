# Search-Navigator — Search Box

**check_types**: [search-box]

## Analysis Criteria

Analyse the screenshot and accessibility tree for:

**Search Box Issues:**
- Search box not visible or hard to find
- Missing search icon or submit button
- Search input field too small
- No placeholder text or unclear purpose
- Autocomplete not working
- Search suggestions displaying incorrectly
- Search button not accessible via keyboard
- No visual feedback when typing
- Search clearing without confirmation
- Mobile search issues (keyboard covering results)
