# Navigation-Expert — Homepage

**check_types**: [homepage]

## Analysis Criteria

Analyse the screenshot and accessibility tree for:

**Homepage Issues:**
- Key navigation elements broken or missing
- Hero section not loading
- Featured content not displaying
- Search functionality broken
- Call-to-action buttons not working
- Logo link not going to homepage
- Slider/carousel not functioning
- Latest content not loading
- Footer links broken
- Mobile menu not working
- Layout broken on different screen sizes
