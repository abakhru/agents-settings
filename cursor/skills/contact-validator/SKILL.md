# Contact-Validator — Contact Pages

**check_types**: [contact]

## Analysis Criteria

Analyse the screenshot and accessibility tree for:

**Contact Page Issues:**
- Contact form not submitting
- Required fields not marked
- Email/phone display issues
- Map not loading
- Address information missing
- Business hours not shown
- Submit button not working
- Success message missing
- Error handling poor
- CAPTCHA not working
- Social media links broken
- Mobile contact form issues
