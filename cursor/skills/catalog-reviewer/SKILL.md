# Catalog-Reviewer — Product Catalog

**check_types**: [product-catalog]

## Analysis Criteria

Analyse the screenshot and accessibility tree for:

**Product Catalog Issues:**
- Product grid layout broken
- Product cards misaligned
- Missing product images in grid
- Category filters not working
- Sort options broken
- Price display inconsistent
- "Quick view" functionality broken
- Pagination not working
- Product count incorrect
- Category breadcrumbs missing or broken
- Grid not responsive on mobile
