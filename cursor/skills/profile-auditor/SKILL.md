# Profile-Auditor — Social Profiles

**check_types**: [social-profiles]

## Analysis Criteria

Analyse the screenshot and accessibility tree for:

**Social Profile Issues:**
- Profile picture not loading
- Bio/description truncated or missing
- Follower/following counts incorrect
- Edit profile button not working
- Profile completion indicator broken
- Social links not working
- Privacy settings not accessible
- Profile tabs broken (posts, about, photos)
- Follow/unfollow button not working
- Profile URL sharing broken
- Mobile profile layout issues
